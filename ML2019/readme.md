## 머신러닝 연구회(2019년)
- 2019년은 빅데이터 연구방법론 중 
- (데이터 수집-저장) ELKv4,
네이버 뉴스, 
- (데이터 분석-처리) 
목적에 따라 머신러닝 연구회와 블록체인 연구회를 구분해서 운영하고자 합니다.
- 머신러닝 연구회는 연구자 활용을
애저 ML 스튜디오를 2회차
캐글스터디를 3회차
- 블록체인 연구회는 연구대상을 블록체인으로 합니다.
하이퍼레저 구축을 목표로 합니다.
